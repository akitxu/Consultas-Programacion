Problemas al convertir archivos csv con Libreoffice.¶

Descargo de Internet ficheros con el formato csv para su conversión posterior en dataframes con Python.

La gráfica de los ficheros csv, se muestra sin problemas.

Si con LibreOffice (version 6.4) convierto los ficheros a los formatos xlsx, xls y ods, las gráficas son incorrectas.

Leer archivo ods¶

In [8]:
import pandas as pd
import ezodf

def read_ods(filename, sheet_no=0, header=0):
    tab = ezodf.opendoc(filename=filename).sheets[sheet_no]
    return pd.DataFrame({col[header].value:[x.value for x in col[header+1:]]
                            for col in tab.columns()})
df = read_ods(filename="/media/enri/Mi_Proyecto/Py_Proyecto_2021/Ficheros_pruebas/Acatis.ods")
df = df.set_index("Date")
